# Flappy Bird - Godot 4 纯代码实现

这是一个使用 Godot 4 纯代码生成的 Flappy Bird 风格游戏。

## 如何运行

1. 打开 Godot 4.3+
2. 点击「导入」
3. 选择此文件夹中的 `project.godot`
4. 点击「导入并编辑」
5. 按 F5 运行

## 操作方式

- **鼠标点击** 或 **空格键**：让小鸟向上飞
- 穿过管道间隙得分
- 碰到管道或地面游戏结束
- 游戏结束后点击重新开始

## 项目结构

```
flappy-bird/
├── project.godot           # 项目配置
├── scenes/
│   ├── main.tscn          # 主场景
│   ├── bird.tscn          # 小鸟场景（备用）
│   └── pipe.tscn          # 管道场景
├── scripts/
│   ├── game_manager.gd    # 游戏管理器
│   ├── bird.gd            # 小鸟控制
│   └── pipe.gd            # 管道逻辑
└── README.md
```

## 核心机制解析

### 1. 小鸟物理 (bird.gd)
- 重力：每帧增加向下速度
- 跳跃：按键时设置向上速度
- 旋转：根据垂直速度旋转小鸟，增加视觉效果

### 2. 管道系统 (pipe.gd)
- 管道向左移动
- 上下管道之间有固定间隙
- 间隙中有隐形计分区域
- 离开屏幕后自动销毁

### 3. 游戏流程 (game_manager.gd)
- READY：等待开始
- PLAYING：游戏进行中，定时生成管道
- GAME_OVER：显示结束界面，等待重新开始

## 可调整参数

在 Godot 编辑器中选中节点，可以在右侧面板调整：

### 小鸟 (Bird)
- `gravity`: 重力大小（默认 1200）
- `flap_strength`: 跳跃力度（默认 -400，负数表示向上）
- `max_fall_speed`: 最大下落速度（默认 600）

### 游戏管理器 (Main)
- `pipe_spawn_interval`: 管道生成间隔秒数（默认 1.8）

### 管道 (Pipe)
- `speed`: 移动速度（默认 200）
- `gap_size`: 间隙大小（默认 200）

## 导出到网页 (itch.io)

1. 菜单：项目 → 导出
2. 添加预设 → Web
3. 下载导出模板（首次需要）
4. 导出项目
5. 上传 HTML 文件到 itch.io
