# Hades Demo - Godot 4 动作游戏

这是一个使用 Godot 4 开发的哈迪斯风格俯视角动作游戏Demo。

## 如何运行

1. 打开 Godot 4.5+
2. 点击「导入」
3. 选择此文件夹中的 `project.godot`
4. 点击「导入并编辑」
5. 按 F5 运行

## 操作方式

| 按键 | 功能 |
|------|------|
| **WASD** | 移动 |
| **Shift / 空格** | 冲刺（带无敌帧） |
| **鼠标左键** | 攻击 |
| **点击** | 开始/重新开始 |

## 游戏玩法

- 在封闭的竞技场中战斗
- 消灭所有敌人以通过当前波次
- 共3波敌人，难度递增（3/4/5个敌人）
- 通关所有波次获得胜利
- 玩家血量归零则游戏结束

## 核心系统

### 1. 玩家系统 (player.gd)
- **移动**: 8方向移动，速度300
- **冲刺**: 0.15秒无敌，速度800，0.5秒冷却
- **攻击**: 扇形范围，伤害20，0.3秒冷却
- **血量**: 100 HP，受伤后0.5秒无敌

### 2. 敌人系统 (enemy.gd)
- **AI**: 追踪玩家，追踪范围500
- **攻击**: 近身攻击，伤害15，1秒冷却
- **血量**: 30 HP
- **死亡**: 淡出动画后销毁

### 3. 波次系统 (game_manager.gd)
- 3波敌人：第1波3个，第2波4个，第3波5个
- 每波敌人清完后2秒开始下一波
- 敌人从房间边缘生成

## 可调整参数

在 Godot 编辑器中选中节点，可在右侧面板调整：

### 玩家 (Player)
- `move_speed`: 移动速度（默认 300）
- `dash_speed`: 冲刺速度（默认 800）
- `max_health`: 最大血量（默认 100）
- `attack_damage`: 攻击伤害（默认 20）

### 敌人 (Enemy)
- `move_speed`: 移动速度（默认 150）
- `max_health`: 最大血量（默认 30）
- `attack_damage`: 攻击伤害（默认 15）

### 游戏管理器 (Main)
- `waves`: 每波敌人数量（默认 [3, 4, 5]）
- `spawn_delay`: 敌人生成间隔（默认 0.5秒）
- `wave_delay`: 波次间隔（默认 2秒）
